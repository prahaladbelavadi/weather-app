# weather-app
A command line weather-app

# Steps: 
1. download git repository
2. run `npm install`
3. run `npm start`
4. hit `Ctrl + C`
5. Run `node -c  <city name>`

## Response

It's 63.34 degrees Farenheit in Boston!
